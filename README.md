# vscode使用技巧

### 快捷键的使用
###### 切换同一编辑器不同的标签页：control+tab
###### 上下移动一行： Option+Up 或 option+Down
###### 向上向下复制一行： Shift+Option+Up 或 Shift+Option+Down
###### 跳到当前行的头部，尾部：Fn+←，Fn+→
###### 插入多光标：Option+click或Cmd+Option+up/Cmd+Option+down
###### 向前缩进当前行：Shift+tab
###### Command + [ 减少缩进
###### Command + ] 增加缩进
###### 在所选行的行尾插入光标：Option + l
